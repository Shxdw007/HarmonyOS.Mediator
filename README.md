# Harmony OS Mediator 🧠🤝

**Harmony OS Mediator** — это AI-ассистент для корпоративных мессенджеров, который в реальном времени анализирует тональность общения сотрудников и помогает снизить уровень токсичности в команде. 

ИИ замечает пассивную агрессию в чате и помогает автору перефразировать текст так, чтобы он звучал профессионально и без негатива.

## 🚀 Ключевые особенности
* **Zero-Data-Leak (Приватность):** Анализ текста происходит локально. Данные корпоративной переписки не отправляются в OpenAI или другие облачные сервисы.
* **Smart Sentiment Analysis:** ИИ понимает контекст, корпоративный сленг и скрытую агрессию, а не просто реагирует на стоп-слова.
* **Мгновенная реакция:** Бот отвечает за доли секунды благодаря асинхронной Webhook-интеграции с Telegram.
* **Анонимная аналитика:** Система сохраняет факты резкого общения (без текста самих сообщений) в локальную базу данных для последующего построения графиков уровня стресса в команде.

## 🛠 Технический стек
* **Backend:** C#, ASP.NET Core Web API
* **Фреймворк:** .NET 10.0
* **База данных:** SQLite + Entity Framework Core
* **AI Engine:** Ollama (локальный хостинг моделей)
* **LLM Model:** Qwen 2.5 (7B параметров) — можно легко заменить на любую другую модель (например, Llama 3.1 или Gemma 2), подходящую под ваше железо.
* **Интеграция:** Telegram.Bot API, ngrok (для проброса туннелей)

## ⚙️ Как запустить локально

1. Установите и запустите [Ollama](https://ollama.com/).
2. Скачайте модель Qwen 2.5, выполнив в терминале: `ollama run qwen2.5:7b`
3. Создайте бота через `@BotFather` в Telegram и получите токен.
4. Пробросьте туннель к вашему локальному порту (например, 5292) с помощью ngrok: `ngrok http 5292`
5. Вставьте HTTPS-ссылку от ngrok и токен бота в файл `Program.cs`.
